# 前言

  1. 为了这个光明而勇敢的世界

  2. 本篇是大陆学校的名单整理，以及相关图片材料，仓促而有限，也未能一一核实，请见谅

  3. 为了这篇整理在墙内存在时间更长，以及整理者本人的安全，有些更激动人心的标语暂时不能上传，但我看到了，我相信更多人也看到了

  4. 整理者本人也在学校有过行动并被处理过，所以深深知道这些勇敢的人会有怎样的遭遇、付出怎样的代价，向你们致敬

  5. 他们并不可怕。让我们铭记，让我们团结
